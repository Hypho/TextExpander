# PAD — Product Spine 产品主干
> 挂载点：/ledger.init 时生成。定义产品目标、业务主流程、核心实体、体验一致性规则和权限模型，pid / contract / build / verify 阶段按需读取。
版本：v1.0 | 最后更新：2026-05-13

> ⚠️ 体积控制：建议不超过 15,000 字符（约 300 行）。

## 产品目标
帮助 Windows 用户通过输入缩写词快速替换为预设文本，提升重复文本输入效率。

## 目标用户与核心场景
| 用户/角色 | 场景 | 目标 | 关键限制 |
|-----------|------|------|----------|
| 文职人员 | 频繁输入地址、邮箱、手机号等固定文本 | 一键扩展，减少重复输入 | 不熟悉技术，需要简单配置 |
| 开发者 | 快速插入代码片段、常用命令模板 | 缩写词触发代码模板 | 需要支持多行文本 |
| 方案架构师 | 常用方案描述、标准化回复 | 快速调用预设文本块 | 需要变量支持（日期等） |

## 核心业务主流程
| Step | 用户目标 | 用户动作 | 系统状态变化 | 对应功能 | 必须体验 | 意图来源 |
|------|----------|----------|--------------|----------|----------|----------|
| S1 | 设置缩写词规则 | 打开管理界面，添加规则 | JSON 配置文件新增条目 | rule-manager-gui | 添加成功提示 | - |
| S2 | 日常输入时触发扩展 | 输入缩写词 + 终止键 | 缩写词被替换为预设文本 | keyboard-hook-engine | 替换瞬间无感知延迟 | - |
| S3 | 暂停/恢复扩展 | 托盘右键菜单切换 | 钩子激活/暂停 | tray-resident | 状态即时切换 | - |

## 功能类型定义
- 主流程功能：直接推进核心业务结果（键盘钩子引擎、规则管理）
- 辅助功能：服务某个主流程节点，但不改变主流程（变量解析）
- 管理功能：配置、维护、运营（托盘驻留、开机自启）

## 核心实体

### Rule（缩写词规则）
| 字段 | 类型 | 说明 | 必填 |
|------|------|------|------|
| id | string (uuid) | 唯一标识 | 是 |
| abbreviation | string | 缩写词（如 ;addr） | 是 |
| replacement | string | 替换文本（支持变量） | 是 |
| enabled | bool | 是否启用 | 是 |
| createdAt | string (ISO 8601) | 创建时间 | 是 |
| updatedAt | string (ISO 8601) | 更新时间 | 否 |

状态机：无状态（enabled/disabled 为开关，非工作流状态）
关联：一条 Rule 对应一条缩写词→文本映射

### AppConfig（全局配置）
| 字段 | 类型 | 说明 | 必填 |
|------|------|------|------|
| terminatorKeys | string[] | 终止键列表（Tab/Space/Enter） | 是 |
| startOnBoot | bool | 开机自启 | 是 |
| enabled | bool | 全局开关 | 是 |

状态机：无

## 体验一致性规则
- 成功后去向：替换完成后光标定位在替换文本末尾
- 空状态：规则列表为空时提示"点击添加按钮创建第一条规则"
- 加载态：应用启动 < 1s 内就绪，托盘图标出现即表示可用
- 错误提示：配置文件损坏时提示"配置文件异常，已恢复为默认设置"
- 重复操作：相同缩写词不允许重复添加
- 取消/返回：编辑规则时可取消，不保存修改

## 全局规则
- 删除策略：硬删除（配置文件直接覆盖）
- 时间格式：ISO 8601
- 状态变更：不记录操作历史（轻量工具，不需要审计）

## 权限模型
| 角色 | 数据范围 | 限制 |
|------|---------|------|
| 本地用户 | 所有规则和配置 | 无额外限制（单用户桌面工具） |

## 明确不做
| 类别 | 内容 | 理由 |
|------|------|------|
| 永不做 | 云端同步、多设备共享 | 超出轻量工具定位 |
| 永不做 | 运行 Quicker 动作 / 调用外部 API | 复制核心文本扩展功能即可 |
| 本版本不做 | 进程绑定（不同软件不同规则） | MVP 不需要 |
| 本版本不做 | 正则匹配 / 模糊匹配 | 缩写词精确匹配足够 |
| 简化做 | 变量仅支持 {date} {time} {clipboard} | 覆盖 80% 场景 |
